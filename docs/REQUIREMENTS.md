## 1. 概述

GreenBite 是一个面向香港市场的本地有机农产品电商平台。用户可以通过网页浏览由本地农场直供的季节性农产品，订阅每周配送计划，并获取 AI 生成的个性化每日菜单。

### 1.1 目标

- 连接香港消费者与本地有机农场，减少食物里程
- 通过订阅模式提供稳定的农产品供应
- 利用 AI 生成个性化膳食建议，降低食物浪费
- 支持中英双语（简/繁/英）

### 1.2 用户角色

| 角色 | 描述 |
|------|------|
| 访客 | 可浏览产品目录、订阅计划，需注册后才能购物 |
| 注册用户 | 加购、下单、订阅、填写饮食问卷、获取 AI 菜单 |
| 商品卖家 | 注册用户上传并管理自己发布的商品 |
| 管理员 | 管理全平台商品、订单、用户权限 |
