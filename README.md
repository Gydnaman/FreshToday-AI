# GreenBite

> A green lifestyle e-commerce platform built with Laravel, jQuery, and MySQL.

## Features

- Local organic agriculture products
- Subscription plans
- Order tracking

## Tech Stack

- Laravel
- jQuery
- MySQL
- Tailwind CSS

## Requirements

- PHP 8.2+
- Composer
- Node.js & npm
- MySQL

## Getting Started

1. Clone the repository
2. Run `composer install`
3. Run `npm install`
4. Copy `.env.example` to `.env` and configure your database
5. Run `php artisan key:generate`
6. Run `php artisan migrate`
7. Start server with `php artisan serve` and `npm run dev`
